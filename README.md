## Hướng dẫn cài đặt

Tại thư mục của project, mở cmd/terminal và nhập:

### `npm install`

Các package cần thiết được cài đặt.\
Sau đó khởi chạy project với câu lệnh:

### `npm start`

Mở [http://localhost:3000](http://localhost:3000) để xem trang web trên trình duyệt.
